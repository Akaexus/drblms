Class: interfaceHarvester

interfaceHarvester~interfaceHarvester

Zbiera i zwraca obiekt zawierający interfejsy widziane przez system i jednocześnie zawarte w konfiguracji DRBL (domyślnie drblpush.conf). Nie potrafi wykryć interfejsu, który nie został jeszcze użyty (nie zostały wysłane/odebrane żadne dane).
